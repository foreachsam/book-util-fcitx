fcitx-bin 套件探索

下載套件

下載「fcitx-bin」這個套件。

$ apt download fcitx-bin

解開套件

解開「fcitx-bin_1%3a4.2.9.1-1ubuntu1_amd64.deb」

$ dpkg -x fcitx-bin_1%3a4.2.9.1-1ubuntu1_amd64.deb fcitx-bin

下載 Source Package

執行

$ apt source fcitx-bin
